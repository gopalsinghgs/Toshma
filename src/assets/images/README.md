# Image Assets

Place the following file in this directory:

| File                   | Source                         | Used in                  |
|------------------------|--------------------------------|--------------------------|
| `ToshmaLabs_Logo.png`  | Toshma Labs brand logo (PNG)   | SplashScreen, ProductScreen |

The logo you shared in chat should be saved here as:
```
src/assets/images/ToshmaLabs_Logo.png
```

React Native auto-discovers `@2x` and `@3x` variants if you provide:
- `ToshmaLabs_Logo.png`      → 1x (72×72 px)
- `ToshmaLabs_Logo@2x.png`   → 2x (144×144 px)
- `ToshmaLabs_Logo@3x.png`   → 3x (216×216 px)
